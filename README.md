# register-node
a register demo by Node Express MongoDB

``` bash
# clone project from github to your computer
    git clone  git@github.com:huahaichuan/register-node.git
    
# enter into project director
    cd register-node
    
# install dependencies
    npm install
    
#start mongo server
    mongod.exe --dbpath D:\learn\register-node\db (your db path)

# run
    node app.js
    
#open in brower        
    localhost/register
